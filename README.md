# Brazilian E-Commerce Performance Analytics Dashboard

## Project Overview

This is an end-to-end Power BI business intelligence project built using the Brazilian E-Commerce Public Dataset by Olist.

The project analyzes e-commerce sales performance, customer behavior, product categories, seller contribution, delivery performance, payment patterns, and customer review satisfaction.

## Project Objectives

The main objective of this project is to build a professional Power BI analytics solution that covers the full BI workflow:

- Business requirement understanding
- Data profiling
- Data cleaning using Power Query
- Data modeling using a star schema
- DAX measure creation
- KPI design
- Interactive dashboard development
- Data validation
- Power BI Service publishing concepts
- GitHub documentation

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel / CSV
- Git and GitHub
- VS Code

## Dataset

The dataset contains multiple CSV files related to Brazilian e-commerce orders, customers, products, sellers, payments, reviews, and geolocation.
